DIV x SPAN — Demonstração Base

Diferença prática entre elementos block-level (DIV) e inline-level (SPAN).

Inline → Block Inline-block Comparativo Código com bugs Correções
📦

DIV (block)

Ocupa toda a largura e inicia em nova linha. Empilha verticalmente.

🧩

SPAN (inline)

Flui no texto sem quebrar linha, ideal para destacar trechos.

🧱

Box Model

Content, padding, border e margin evidenciados por cores/bordas.

SPAN no meio do parágrafo

Texto antes, trecho inline destacado, texto depois — tudo na mesma linha.
